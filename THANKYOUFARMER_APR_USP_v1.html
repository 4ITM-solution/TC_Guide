PRODUCTION METHOD
  1. Film bare skin close-up before any skincare
  2. Use natural or soft window light — no flash
  3. Tilt the bottle and pour toner directly onto face — catch the milky drip
  4. React naturally to the texture
  5. Cut to absorption shot, then glow reveal
SCENE
Milky Toner Pour
DIRECTION
Tilt the bottle and pour the toner directly onto the face — let it run down the cheek in a thick, milky stream. The texture should be visible and satisfying. Film from a slightly elevated angle so the pour and skin are both in frame.
